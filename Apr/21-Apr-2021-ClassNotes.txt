---------------------------------------------------------------------------------
-> Spring Boot is an approach to develop Spring Based applications with minimal or less configurations.

Spring Boot Advantages
-----------------------
1) Starter Poms
2) Auto Configuration
3) Embedded Servers
4) Acutators
5) Embedded Databases
6) Rapid Application Development

Spring Boot Dis-Advantages
---------------------------
-> Migrating Spring project into Spring Boot application is a difficult task

Conclusion
----------

Spring Boot = (Spring Framework - XML Configuration ) + Auto Configuration + Embedded Servers + Actuators
